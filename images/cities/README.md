# City Images

Place city images in this directory with the naming convention:
- `tokyo.jpg`
- `paris.jpg`
- `new-york.jpg`
etc.

## Image Specifications

### Required Format
- **Format**: JPG (JPEG)
- **Naming**: Lowercase city name with hyphens (e.g., `new-york.jpg`, `hong-kong.jpg`)

### Recommended Dimensions
- **Size**: **1600 x 900 pixels** (16:9 aspect ratio)
- **Minimum**: 1280 x 720px
- **Maximum**: 2560 x 1440px (for high-DPI displays)

### Quality Guidelines
- **File Size**: Keep under 300-500KB per image
- **Compression**: Use 80-85% quality for optimal balance
- **Color Space**: sRGB
- **Orientation**: Landscape (horizontal)

### Content Recommendations
- Feature iconic landmarks or cityscapes
- Use high-quality, well-lit photos
- Avoid heavy text overlays
- Ensure the subject is clearly visible

## Free Image Sources

You can obtain free, high-quality city images from:
- [Unsplash](https://unsplash.com) - Free high-resolution photos
- [Pexels](https://pexels.com) - Free stock photos
- [Pixabay](https://pixabay.com) - Free images and videos

## Image Optimization

Before adding images, optimize them using:
- [TinyJPG](https://tinyjpg.com) - Online compression
- [ImageOptim](https://imageoptim.com) - Mac app
- [Squoosh](https://squoosh.app) - Web-based optimizer
